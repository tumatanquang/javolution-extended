Provides a comprehensive set of high-performance concurrent utility classes, based on the original util.concurrent library by Doug Lea. These classes are optimized for real-time and embedded systems, featuring reduced object creation and compatibility with RTSJ virtual machines.

Overview:

This package contains several categories of concurrent programming utilities:

  • Executors and Thread Pools: Classes like {@link javolution.util.concurrent.PooledExecutor PooledExecutor}, {@link javolution.util.concurrent.ThreadedExecutor ThreadedExecutor}, and {@link javolution.util.concurrent.QueuedExecutor QueuedExecutor} for managing thread execution and asynchronous task processing.
  • Synchronizers: Mutual exclusion and coordination mechanisms such as {@link javolution.util.concurrent.Semaphore Semaphore}, {@link javolution.util.concurrent.Mutex Mutex}, {@link javolution.util.concurrent.Latch Latch}, {@link javolution.util.concurrent.CountDown CountDown}, and {@link javolution.util.concurrent.CyclicBarrier CyclicBarrier}.
  • Concurrent Collections: High-throughput collections including {@link javolution.util.concurrent.ConcurrentHashMap ConcurrentHashMap}, {@link javolution.util.concurrent.ConcurrentReaderHashMap ConcurrentReaderHashMap}, and {@link javolution.util.concurrent.CopyOnWriteArrayList CopyOnWriteArrayList}.
  • Queues: Thread-safe channel and queue implementations such as {@link javolution.util.concurrent.LinkedQueue LinkedQueue}, {@link javolution.util.concurrent.BoundedLinkedQueue BoundedLinkedQueue}, and the non-blocking {@link javolution.util.concurrent.WaitFreeQueue WaitFreeQueue}.
