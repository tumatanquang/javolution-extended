Provides high-performance, resizable list implementations for Java primitive types, avoiding object creation and auto-boxing overhead.

Overview:

This package contains dedicated classes for all Java primitive types:

  • {@link javolution.util.primitive.FastBooleanArrayList FastBooleanArrayList} for boolean
  • {@link javolution.util.primitive.FastByteArrayList FastByteArrayList} for byte
  • {@link javolution.util.primitive.FastCharArrayList FastCharArrayList} for char
  • {@link javolution.util.primitive.FastShortArrayList FastShortArrayList} for short
  • {@link javolution.util.primitive.FastIntArrayList FastIntArrayList} for int
  • {@link javolution.util.primitive.FastLongArrayList FastLongArrayList} for long
  • {@link javolution.util.primitive.FastFloatArrayList FastFloatArrayList} for float
  • {@link javolution.util.primitive.FastDoubleArrayList FastDoubleArrayList} for double

All primitive array lists in this package are optimized for speed, predictability, and minimal memory footprint, making them suitable for real-time applications and memory-constrained environments.
