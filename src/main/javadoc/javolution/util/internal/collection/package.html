Contains internal implementation classes for high-performance collections.

Important Note:

Classes in this package, such as {@link javolution.util.internal.collection.FastSharedCollection FastSharedCollection} and {@link javolution.util.internal.collection.FastUnmodifiableCollection FastUnmodifiableCollection}, are internal implementation details of the Javolution library. They are not intended for direct use by application developers. Instead, use the standard wrapper methods like FastCollection.unmodifiable() or FastCollection.shared().
